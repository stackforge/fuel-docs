Mirantis OpenStack Demo & Tutorial

Release 6.1 of Mirantis OpenStack provides the latest version (Juno) of the OpenStack open cloud framework in robust, hardened configurations, auto installable via Fuel on virtual or physical hardware (you can even build a cloud on your laptop).

Release 6.1 also features a Fuel plugin architecture (that lets you easily add VPNaaS, LBaaS, GlusterFS and NetApp storage support), improved HA, enhanced networking, compatibility and much more.
