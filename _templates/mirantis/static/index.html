Mirantis OpenStack is made up of three components:

  • Hardened packages in Mirantis OpenStack. Hardened packages in Mirantis OpenStack include the core OpenStack projects, updated with each stable release of OpenStack, and supporting a broad range of operating systems, hypervisors, and deployment topologies.
  • Fuel. Fuel is an open source, software life cycle management application that deploys multiple OpenStack environments from a single interface and then enables you to manage those environments post deployment.
  • Support. A complementary 30-day basic trial support subscription is available.

Mirantis OpenStack Demo & Tutorial

Release 6.0 of Mirantis OpenStack provides the latest version (Juno) of the OpenStack open cloud framework in robust, hardened configurations, auto installable via Fuel on virtual or physical hardware (you can even build a cloud on your laptop).

Release 6.0 features a new Fuel plugin architecture (that lets you easily add VPNaaS, LBaaS, GlusterFS and NetApp storage support), improved HA, enhanced L2/L3 networking, full-spectrum VMware (vCenter+NSX+VDMK) compatibility and much more.
