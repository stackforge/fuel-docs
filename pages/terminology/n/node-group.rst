Node group
----------

A node group is a set of nodes,
grouped by networks;
a cluster is a set of node groups.
This is a :ref:`nailgun-term` concept
that is introduced to support the
:ref:`Multiple Cluster Networks<mcn-arch>` feature
in Fuel 6.0 and later.
