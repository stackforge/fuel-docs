Hardware and System monitoring
==============================

The hardware and system monitoring must be perform on each node composing the OpenStack cluster.

Hardware
--------

IPMI:

- temperature
- fan
- voltage
- memory ecc

disk health (HDD tools)

Operating System
----------------


disk space
inode usage

CPU usage:

- User/System/IO wait
- TOP processes

load average

RAM Usage

Swap Usage

RAID health
