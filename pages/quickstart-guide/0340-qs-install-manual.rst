Installing Mirantis OpenStack Manually
======================================

If you do not want to use the Mirantis VirtualBox scripts, you can install 
Mirantis OpenStack on VirtualBox manually. However, Mirantis recommends to use 
the manual installation instructions for evaluation only.
 
.. warning:: Do not use a manually configured environment for production or 
			 customized deployments.
