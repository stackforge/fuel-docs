Hardware requirements
~~~~~~~~~~~~~~~~~~~~~

Planning hardware requirements for an OpenStack environment is a complex task
that involves analysis of applications that you plan to run in your cloud, as
well as understanding how your cloud will expand over time. Therefore, in
each use case hardware requirements for Fuel Target nodes will differ.
However, hardware requirements for Fuel Master node are identical for all
deployments.
