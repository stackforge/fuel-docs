Preface
=============================

This documentation provides information on how to use Mirantis Fuel 
to deploy OpenStack environment. The information is for reference purposes 
and is subject to change.

Intended Audience
-----------------------------

Intended for OpenStack administrators or "DevOps" (development operations)
professionals, this documentation assumes that you have experience with
network and cloud computing concepts.
